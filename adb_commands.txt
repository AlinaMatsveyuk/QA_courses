 1. Отобразить подключённый девайс в консоли.

 C:\adb>adb devices

 2. Вывести адрес приложения todolist в системе Android

 C:\adb>adb shell "pm list packages -f | grep todolist"

 3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB

 C:\adb>adb install -t D:\Android\001_ToDoList-master\001_ToDoList-master\app\build\outputs\apk\debug\app-debug.apk

 4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде.

 C:\adb>adb shell screencap /storage/emulated/0/DCIM/Screenshots/screen_1.png && adb pull /storage/emulated/0/DCIM/Screenshots/screen_1.png 
 D:\Android\001_ToDolist-master\screen_2.png

 5. Вывести в консоль логи приложения todolist
 6. Скопировать логи приложения todolist на компьютер.

 C:\adb>adb logcat | findstr com.android.todolist >todolist.log
 C:\adb>start . todolist.log

 7. Удалить приложение todolist с телефона через ADB

 C:\adb>adb uninstall com.android.todolist
